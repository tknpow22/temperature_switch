LCD表示内容
===========

設定時ほか
----------

#YYYY/MM/DD HH:mm@XTT
	// (現在日時)YYYY=年/MM=月/DD=日 HH=時:mm=分
	// @=リセット動作X=モードTT=現在の温度設定
	//   リセットなし(スペース)、リセットあり(@)、リセット中(R)、モードは自動(A)、手動(M)
	//   手動は指定した温度から変更しない。
	//   自動は設定通りに温度変更する。
	//   下記値の設定時はモードに X を表示する。
	//   リセットについては後述

#SR HH:mm SS HH:mm
	// SR(日の出時刻) HH=時:mm=分 SS(日の入り時刻) HH=時:mm=分

#ASTT ETT PSTT ETT
	// A STT=午前の初期温度 ETT=午前の最終温度
	// P STT=午後の初期温度 ETT=午後の最終温度

#ASHm-HH:mm EHm-HH:mm
	// 午前の温度処理用
	// S(日の出から処理開始までの時間と時刻)H=時間m=分(10分単位)-HH=時:mm=分
	// E(処理開始から処理終了までの時間と時刻)H=時間m=分(10分単位)-HH=時:mm=分

#これ以降は設定時のみ表示される

#PSHH:mm EHm-HH:mm
	// 午後の温度処理用
	// S(午後温度の開始時刻)HH=時mm=分
	// E(処理終了から日の入りまでの時間(分))H=時間m=分(10分単位)-HH=時:mm=分

#RX HN MM SNN ENN
	// R(リセット)X=N|F|Dリセットパターン HN=リセットを行う時間間隔(時) HM=リセットする時間(分) S(リセットから除外する開始時)N=時 E(リセットから除外する終了時)N=時
	//   巻き上げ式の天窓では、開閉を続けていくうちに正しい開度ではなくなっていく。
	//   一旦、全開もしくは全閉した後では、ある程度正しい位置になるが、やはり開閉を続けていくと開度がずれていく。
	//   時々、全開もしくは全閉することで位置をリセットすると良いのだが、全開は難しいので、設定温度を最大にあげることで全閉させ、
	//   リセットを行う機能を作った。
	//   この機能を有効にした場合、自動モードのとき、リセットを行う時間間隔(時)で、正時に、リセットする時間(分)だけリセット(最大温度に設定)する
	//   リセットパターンは N:しない、F:終日、D:日の出から日の入りまで
	//   Dを選んだ場合、日の出時刻を基準として行い、日の出直後にはリセットしない
	//   なお、左から5番目のボタンを長押しすると即時リセットを行う。

#LAT IXXXX.D YY YY
	// IXXXX=緯度整数部.D YY=緯度小数部上位2桁 YY=緯度小数部下位2桁

#LNG IXXXX.D YY YY
	// IXXXX=経度整数部.D YY=経度小数部上位2桁 YY=経度小数部下位2桁

時刻設定時
----------

# YYYY MM DD HH mm ss
	// YYYY=年 MM=月 DD=日 HH=時 mm=分 ss=秒

# X
	// X=YまたはN
	//   Y にして完了すると指定した時刻を設定する

サーボ設定時
------------

# TXX:  NNNN
	// TXX=最小温度 NNNN=サーボモーターの温度最小時のパルス値

# TXX:  NNNN
	// TXX=最大温度 NNNN=サーボモーターの温度最大時のパルス値


左から1番目のボタン
===================
#押下で自動モードに移行
#長押しで日時設定へ移行
#日時設定時にさらに長押しでサーボ設定へ移行
#自動モードでリセット中に日時設定やサーボ設定に移行するとリセット処理をキャンセルする

左から2番目のボタン
===================
#ダウンボタン

左から3番目のボタン
===================
#アップボタン

左から4番目のボタン
===================
#設定モードに移行
移行後は、押すたびに設定項目を変更する
#自動モード、手動モードでリセット中に押すとリセット処理をキャンセルする
なお、自動モードのときはそのまま設定モードに移行するので注意。
※他に適当なボタンがなかったので仕方なく割り当てた。

左から5番目のボタン
===================
#設定モードを完了する
その後自動モードに移行
#長押しで即時リセット

注意事項
========
#リセット処理は、内部変数のみで実現しているため、再起動するとキャンセルされる。
再起動にはウォッチドッグタイマによるフリーズに対する再起動も含まれる。
